# Mon Projet

Moteur de recherche basé sur le paradigme de programmation orienté objet utilisant des matrice tf_idf et la similiraté cosinus entre vecteurs


## Description

Ce projet de cours de master informatique contient les différents fichiers permettant de construire un moteur de recherche  
## Prérequis

- Git installé sur votre machine
- Python 3.8+


## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/votre-projet.git
   ```

2. Installez les dépendances (recommandé via un environnement) :
   ```bash
   pip install -r requirements.txt
   ```

## Utilisation

Exécutez le projet avec :
```bash
exécutez le notebook ipynb
```
